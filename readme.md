# `.yawn`

**A tiny open file system for seeing the frame you are inside.**

The basic problem is simple: your mind is trying to hold too much at once.

Attention, memory, emotion, habit, belief, body signal, relationships,
constraints, tools, AI agents, and next actions all show up together. Then you
still have to answer:

```text
what is happening?
what am i inside?
what is missing?
what can i safely try?
what would prove anything changed?
```

`.yawn` gives that inner interface somewhere simple to land.

## Use It Like Madlibs

YAWN is a mirror. You use it by filling out `.yawn` files:

```text
i am observing ___
the current frame is ___
the possible frame is ___
what is missing is ___
the smallest safe move is ___
the boundary is ___
proof would look like ___
afterward, what changed was ___
```

You can fill one out alone, with another person, or with any AI. The AI can help
you mirror the frame, ask clearer questions, make the move smaller, and find
proof gaps.

The point is not to outsource your selfhood.

The point is to interface better with yourself.
